JS - V 02 - Objet littéral

Un objet est un ensemble cohérent de propriétés et de méthodes.

A- Création d'un objet littéral

  • Créer un objet littéral
  • Utiliser le constructeur object()
  • Utiliser une fonction constructuer personnalisée
  • Utiliser le méthode create()

1- Un objet littéral

cf. page d'introduction et le programme

Dans l'exemple précédent on dira qu'on créé un objet nommé "utilisateur".

2- Le caractère, le point "."

Le point "accesseur" qui permet "d'accéder à".

Il va permettre d'accéder aux valeurs de nos différents "membres" mais également de les modifier.

3- Utilisation de this

Le mot-clef this est un prète-nom, il permet de ne pas répéter le nom de l'objet dans l'objet

4- Utilisation des crochets []

Les crochets seront utiles pour les "tableaux" (array) pour appeler une valeur d'une propriété array (et pour construire un tableau).
