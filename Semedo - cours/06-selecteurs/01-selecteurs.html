Les sélecteurs en JS

Ce titre sera sélectionné dans la console

Dans la console nous voyons les différentes méthodes de sélectionner cet élément dont l'Id est titreNiveau2

  1. Dans une variable qui se trouve dans window.onLoad() /// var select1 = titreNiveau2
  2. Dans une variable simple var select2 = titreNiveau2;
  3. Avec getElementById directement dans le document document.getElementById('titreNiveau1');
  4. Avec getElementByClassName je récupère un ensemble, une collection de class document.getElementById('titreNiveau2');
  5. Avec getElementByTagName je récupère un ensemble d'élément par leur nom
