Soit la boucle for dans le .js 13-parite.js; ce programme affiche les nombres pairs jusqu'à 10 dans la console. Tapez le et examinez le résultat dans votre console.

for (var i = 1; i<= 10; i++) {
if(i % 2 === 0) {
console.log(i + ' pair');
}
}

Ce programme utilise le modulo (reste de la division est égal à 0) pour trouver les nombres pairs en dessous de 10.

Exo 1 : Modifiez le programme, dans une boucle for, afin qu'il affiche dans la console, les nombres pairs et impairs.

Exo 2 : Modifier le programme avec un prompt pour récupérer un nombre entier et afficher la nature des 10 nombres supérieurs

Exo 3 : faites la même boucle, demandant un second nombre avec while.
