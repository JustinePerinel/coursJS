Valeurs

Observez le code ci-dessous et tentez de prévoir les valeurs finales de chaque variable à la fin de chaque ligne
(par exemple après la ligne 'var a = 2;' si je faisais un console.log() de a j'aurai 2 comme résultat. cf.la capture d'écran)

L'idée est de faire des calculs vous-mêmes, sur une feuille par exemple AVANT de regarder dans la console !

  • var a = 2; // résultat : a = 2
  • a = a - 1; // résultat : 2 - 1 = 1
  • a++; // résultat : 1 + 1 = 2
  • var b = 8; // résultat : b = 8
  • b += 2; // résultat : 8 + 2 = 10
  • var c = a + b * b; // résultat : 2 + (10*10) = 102
  • var d = a * b + b; // résultat : 2 * 10 + 10 = 30
  • var e = a * (b + b); // résultat : 2 * (10+10) = 40
  • var f = a * b / a; // résultat : 20 / 2 = 10
  • var g = b / a * a; // résultat : (10/2) * 2 = 10

Vérifiez vos prévisions en crant ce programme puis en déboguant pour observer les valeurs des variables.
