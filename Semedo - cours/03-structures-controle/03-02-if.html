1 - If...

La condition if est l'une des plus utilisées et est également la plus simple à appréhender puisqu'elle va nous permettre
d'exécuter un bloc de code SI et SEULEMENT SI le résultat vaut true.

Des if exécutés dans le if.js

  • if(x>1){

    alert('x contient une valuer supérieure à 1');

    }

  • if(x == y){

    alert('x et y contiennent la même valeur');

    }

  • if(y){

    alert('la valeur de y est évaluée à true');

    }

!! Dans ce 3è if : tout test d'une condition va être évalué dans un contexte boléen ; donc à true, à l'excéption des valeurs suivantes qui vont être évaluées à false.

  • Le booléen false
  • La valeur 0
  • Une chaîne de caractère vide
  • La valeur null
  • La valeur undefined
  • La valeur NaN (Not a Number)

2 - Inverser la valeur logique d'un test

Dans un if, le code placé dans le if n'est exécuté que si le résultat de la comparaison est false.

Dans certaines situations, on préfère vérifier si la comparaison est false.

On peut vérifier que c'est faux en comparant explicitement le résultat de la comparaison à false.
