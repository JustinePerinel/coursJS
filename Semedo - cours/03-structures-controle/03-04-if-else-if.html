1 - if... else... if... en JS

La condition if... else... if... (si... sinon... si) est une structure conditionnelle encore plus complète puisqu'elle permet de générer et de prendre en charge autant de cas qu'on le souhaite.

Nous allons pouvoir ajouter autant de else if que l'on veut dans cette condition.


if(x > 1){

alert('x contient une valeur strictement supérieure à 1');

}else if(x == 1) {

alert('x contient la valeur la valeur 1');

}else{

alert('x contient une valeur strictement inférieure à 1');

}

Dès qu'un test va être validé, le js va ignorer les tests suivants. Il faut donc faire bien attention à l'ordre des else if lors de l'écriture d'une condition pour obtenir le résultat souhaité.
