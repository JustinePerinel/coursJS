Les structures conditionnelles ternaires (ou ternaires) sont une autre façon d'écrire nos conditions en utilisant une syntaxe basée sur l'opérateur ternaire ?:.

On peut écrire la condition sur une seule ligne !

1 - Exemples d'utilisation

Les ternaires se présentent sous la forme suivante test ? code à exécuter si true : code à exécuter si false.
