Les diaporamas en JS

1- Un 1er diaporama, avec des photos de plages...

1ère façon de faire, sur les images directement je mets un onclick...

Dans la balise img, je mets cet attribut : onclick="this.src='img/e.jpg';this.alt='une autre plage';"

2- Un 2nd diaporama...

Ce second diaporama est fait avec une fonction anonyme dans un écouteur d'événement, dans les scripts en bas de page.

document.getElementById('btnDiapo2').addEventListener('click', function() {
document.getElementById('imgDiapo2').src = 'img/e.jpg';
document.getElementById('imgDiapo2').alt = 'la piscine est belle';
})
