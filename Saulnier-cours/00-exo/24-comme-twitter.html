Un exo avec Boodstrap 4 et JS

Comme Twitter -

Un script qui contrôle que nous avons écrit 140 caractères et pas plus

Consignes

1

L'utlisateur peut saisir un texte de 140 caractères dans un textarea #tweetContenu

Un paragraphe #compteur affiche les caractères restants

2

  1. Détecter l'évènement à chaque fois que l'utilisateur presse une touche.
  2. Dans la fonction lancée lorsque "l'event" est détecté, calculer le nombre de caractères restants et l'afficher dans le p#compteur

Indications

  • Utiliser l'event keyup (chercher à quoi il peut bien servir)
  • Utiliser la propriété elt.value (elt = élèment) pour récupérer la valeur d'un input ou d'un textarea
  • Utiliser .length pour compter le nombre de caractères d'une chaîne de caractères
  • Utiliser elt.textContent=''; pour remplacer le contenu d'un élément HTML
  • La sélection d'élément avec querySelector - getElementbyId
  • Le gestionnaire d'évènements avec addEventListener
  • La manipulation du DOM avec innerHTML - classList
