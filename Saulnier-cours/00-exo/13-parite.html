Exo 13 - Parité

Soit la boucle for dans le .js 13-parité.js; ce programme affiche les nombres pairs jusqu'à 10 dans la console. Tapez le et examinez le résultat dans votre console.

for (var i = 1; i <=10; i++) {
if ( i % 2 === 0 ) {
console.log(i + ' est pair');
}
}

Ce programme utilise le modulo (reste de la division est égal à 0) pour trouver les nombres pairs en dessous de 10.
