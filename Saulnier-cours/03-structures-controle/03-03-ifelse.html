1 - If... Else... en JS

La structure condtionnelle if... else (en français si... sinon) sera plus complète car on va exécuter un bloc de code si un test renvoie true ou un autre bloc de code dans le cas contraire.

If... Else...

  • if(x >1){
    alert('x contient une valeur supérieur à 1');
    }else{
    alert('x contient une valeur inférieure à 1');
    }
