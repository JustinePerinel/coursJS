JS - III 01 Les structures de contrôle

1 - Les conditions et les boucles

Il existe deux types de structures de contrôle de base, que l'on retrouve dans tous les langues de programmation, et donc en JS

  • Les structures de contrôle conditionnelles : conditions
  • Les structures de contrôle de boucle : les boucles

Les conditions vont nous permettre d'éxécuter un certain nombre d'instruction si et seulement si une certaine condition est vérifiée

Les boucles vont nous permettre d'écécuter un bloc de code en boucle, tant qu'une condition donnée est vérifié

A - Présentation des conditions

  • La condition if (si) ;
  • La condition if... else (si... sinon);
  • La condition if... elseif... else (si... sinonsi... sinon).

Dans le cas où la comparaison est vérifiée, le JavaScript renvoie le booléen true. Dans le cas contraire, le booléen false est renvoyé

Le JavaScript va automatiquement comparer la valeur à gauche de l'opérateur à celle de droite selon l'opérateur de comparaison fourni et renvoyer le boléen true si la comparaison est validée et false si la comparaison est ne l'est pas.
