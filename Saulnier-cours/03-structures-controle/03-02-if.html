Js III - 02 If

1 - If...

La condition if est l'une des plus utlisée et également la plus simple à appréhender puisqu'elle va nous permettre d'éxecuter un bloc de code SI et seulement Si le résultat vaut true.

2 if exécutés dans le if.js

  • if (x > 1){
    }
  • if(x == y) {
    alert('x et y contiennent la même valeur');
    }
  • if(y){
    alert('la valeur de y est évalué à true');
    }

Dans ce 3ème if : tout test d'une condition va être évalué dans un contexte booléen ; donc à true à l'exception des valeurs suivantes sui vont être évaluées à false.

  • Le booléen false
  • La valeur 0
  • Une chaîne de caractère vide
  • La valeur null
  • La valeur undefined
  • La valeur NaN (Not a Number)

2 - Inverser la valeur logique d'un test

Dans un if le code placé dans le if n'est exécuté que SI le résulat de la comparaison est true.

Dans certaines situations, on préfère vérifier si la comparaison est false.

On peut vérifier que c'est faux en comparant explicitement le résultat de la comparaison à false.
