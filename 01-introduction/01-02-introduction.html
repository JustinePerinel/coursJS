I. 02 - Du JS dans body



Placement et action du JS

Ici, le JavaScript est placé directement dans <body>, puis dans les différentes balises des éléments auxquels on veut appliquer du JS. Il va falloir après définir l'action et le résultat que l'on vzeut obtenir. Pas une méthode recommendée comme le CSS inline, rare et c'est mieux de ne pas le faire.

On a définit ici l'action d'alerte quand on clique sur le premier bouton. Une fenêtre apparaît avec un message choisi dans le code. Sur le deuxième bouton, un paragraphe est ajouté dans body lorsqu'il est cliqué.
