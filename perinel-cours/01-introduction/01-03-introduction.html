I. 03 - Du JS dans l'en-tête et dans body

On va pouvoir placer n'importe où dans notre code, aussi bien dans la balise head qu'au sein de l'émémént body mais, contrairement à la page précedente, le code est icic placé dans une balise <script>. On place également plusieurs bouts de code JS à différents endroits de la page.

Placement et action du JS

Ici, le JS permet de, sur le premier bouton, déclancher une alerte: "Coucou c'est encore moi !" et sur le deuxième de créer un paragraphe dans body "La grand-mère de Patrick fait du vélo". Le code pour l'alerte est dans la partie head, dans une balise <script> tandis que le code pour le deuxième bouton se trouve dans body dans une balise <script>, en-dessous des boutons.
