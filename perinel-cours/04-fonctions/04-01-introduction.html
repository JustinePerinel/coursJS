IV. 01 Les fonctions

Les fonctions en js sont incontournables ! En voici deux exemples et une définition.

1 - Les fonctions prédéfinies

Une fonction est un bloc de code nommé et réutilisable, et dont le but est d'effectuer une tâche précise ; les fonctions possèdent de nombreux atouts.

Exemple random()

Il existe des fonctions prédéfinies, telle que random() qui est une fonction mathématique. Ici, un code qui permet de générer un nombre compris entre 0 et 100.

Un code qui permet de mettre un nombre, ici un nombre contenu entre 0 et 10.

Un code qui permet de mettre un nombre, ici un nombre décimal contenu entre 0 et 1.

Exemple replace()

Ici, on utilise la fonction replace(), une fonction "string", qui appartient à l'objet string et qui va permettre de remplacer une partie d'une variable. Le premier argument de replace() estr la chaîne de caractères que l'on cherche et le second est celle qui va la remplacer.

2 - Les fonctions personnalisées

Exemples :

function randomX100() {
return Math.random() * 100;
}

document.getElementById('p3').innerHTML = 'Je génère un nombre compris entre 0 et 100 : ' + randomX100();

    3 - Récapitulatif sur les fonctions

      Les fonctions sont des blocs de code nommés et réutilisables et dont le but est d'effectuer une tâche précise.

    • Il existe deux types de fonctions, les fonctions natives ou prédéfinies (ou des méthodes) que l'on aura qu'a appeler et les fonctions personnalisées que l'on va créer.
    • ¨Pour exécuter le code d'une dfonction il faut l'appeler. Pour cela, il suffit d'écrire son nom suivi de parenthèses, en y passant éventuellement des arguments.
    • On crée une fonction personnalisée grâce au mot-clef 'function':
      Si une fonction a besoin qu'on lui passe des valeurs pour fonctionner, alors on définira des paramètres lors de sa définition.
      Lors de son appel, on lui passera des arguments qui prendront la place des paramètres.
