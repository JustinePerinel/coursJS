II. 03 Les types de données JavaScript

7 types de données JavaScript

  1. 1 - String ou"chaîne de caractères"

    En JavaScript, on va pouvoir utiliser indifférement des guillemets doubles ou des quotes pour entourer une chaîne de caractères. Ces méthodes vont être strictement équivalentes, à la différence d'autres langages comme le PHP, par exemple.

    Faites attention cependant à un point : si votre chaîne contient un caractère qui est le même que le délimiteur de chaîne choisi, il faudra neutraliser ce caaractère en l'échappant au moyen d'un antislash, ou changer de délimiteur.


    Grâce à ce code JS qui se trouve dans la page type.js, le résultat suivant va s'afficher. On demande au navigateur d'afficher Type de données pour la variable présentation puis d'indiquer quel est le type de cette variable : ici string car la variable contient une chaine de caractère "Je m'appelle Michaël".

    document.getElementById('p1').innerHTML = 'Type de données pour la variable présentation : ' + typeof presentation;

  2. 2 - Number ou nombres

    Les variables en JavaScript vont également pouvoir stocker des nombres. Des nombres entiers positifs ou négatifs ou a virgule (décimaux).

    Grâce à ce code JS qui se trouve dans la page type.js, le résultat suivant va s'afficher. On demande au navigateur d'afficher Type de données pour la variable age puis pour la vaiable code postal puis d'indiquer quel est le type de cette variable : ici Number car ces variables contiennent des nombres : 25 et 92150.

    document.getElementById('p2').innerHTML = 'Type de données pour la variable age : ' + typeof age;
    document.getElementById('p3').innerHTML = 'Type de données pour la variable code postale : ' + typeof codePostal;

  3. 3 - Boolean ou booléen

    Ce type de données sera utile poour vérifier si un test est vrai ou faux.

    Un booléen, en algèbre, est une valeur binaire (soit 0, soit 1). En informatique, le type booléen est un type qui ne contient que deux valeurs : les valeurs true (vrai) et false (faux).

    let test = 8 > 4;
    document.getElementById('b').innerHTML = 'Type de données pour la variable test : ' + typeof test;

  4. 4 - Null ou vide.

    Ce sont des types un peu particuliers.

  5. 5 - Undefined ou indéfinies

    Si on déclare une variable sans lui attribuer une valeur, alors son type sera undefined.

    let pomme;
    document.getElementById('p5').innerHTML = 'Type de données pour la variable pomme : ' + typeof pomme;

    Ici, le code est indéfini car la valeur de pomme n'a pas été définie. Si le code avait été let pomme = 'reinette';, le type aurait été string.

  6. 6/7 - Symboles et object

    Les objets sont des structures complexes qui vont pouvoir stocker plusieurs valeurs en même temps et que nous étudierons plus tard car il s'agit là d'un sujet complexe à appréhender pour un débutant.

    Si on déclare une variable et qu'on lui passe null, alors son type sera Object.
