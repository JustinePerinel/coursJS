II. 05 - La concatenation et les littéraux de gabarit

1 - La concaténation JS

Littéralement, concaténer signifie "mettre bout à bout". L'opérateur de concaténation est le +. On utilisera notamment la concaténation pour intégrer le contenu de variable dans une page en html.

On utilisera par exemple le code suivant :
let monAge = 27;
let phrase = 'Bonjour, j\'ai ';
document.getElementById('p1').innerHTML = phrase + monAge + ' ans';
pour avoir ce résultat :

Ici, on concatène la variable phrase qui contient Bonjour j'ai et la variable monAge qui contient 27 et enfin la chaîne de caractère 'ans' pour créer une phrase et l'intégrer dans le HTML. On remarque aussi dans le code le caractère d'échappement \ pour permettre à l'apostrophe d'être considérée comme telle.

Atention, utilisé avec deux chiffres le signe + sert à additionner.

2 - Les littéraux de gabarits

IL faut toujours entourer les chaînes de caractères ou 'string' avec des apostrophes ' ' ou des guillemets doubles " ". Il existe une nouvelle et troisième manière de les entourer : on utilisera l'accent grave `` : alt gr + 7.
