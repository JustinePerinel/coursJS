II. 02 - Des exemples de variables

1. Deux exemples de variables

Sur cette page, nous avons vu deux types de variables et la façon dont elles s'écrivent.

A - Let, sur une ligne ou deux

let permet de déclarer une variable sur le bloc auquel il appratient. On peut déclarer la variable de deux façons :

  • En 2 lignes, en déclarant d'abaord la variable et en lui assignant ensuite un contenu
    let variable;
    variable = ''; EXEMPLE
    let monAge;
    monAge = 27;
  • En une ligne
    let variable = ''; EXEMPLE
    let ville = 'Suresnes';

B - Alert, faire apparaître une fenêtre d'alerte

Lorsque l'on a déclaré une variable ou que l'on veut faire apparaître une alerte sur la page web, il est possible de la déclarer en JavaScript. 2 solutions existent alors:

  • Lorsque l'on a déclaré une variable comme suit :
    let prenom = 'Justine';
    et que l'on déclare ensuite
    alert(prenom);
    l'alerte affichera à ce moment là le contenu de la variable prenom, c'est-à-dire Justine.
  • On peut aussi déclarer directement ce que l'on veut entre les parenthèses de l'alerte :
    alert('Bienvenue chez les Ch\'tis');
    les apostrophes autour de la phrase permettent d'indiquer au navigateur que l'on veut qu'il affiche ce qui se trouve dans les parenthèses.

2. console.log ou console.info : afficher le contenu des variables dans l'inspecteur

Dans du JS (dans la balise script ou dans une page externe), console.log(prenom) ou console.info(prenom) vont permettre d'afficher le contenu de la variable indiquée entre parenthèses dans la partie console de l'inspecteur(F12 sur PC). On peut indiquer plusieurs variables dans le console.log en les séparant pas une virgule et un espace.
