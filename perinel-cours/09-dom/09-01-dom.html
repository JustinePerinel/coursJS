JS - IX 01 Le DOM

Document Object Model

Définition

1 - Le DOM est une interface de programmation pour des documents HTML ou XML qui représente le document (la page web actuelle) sous une forme qui permet aux langages de script comme le JavaScript d'y accéder et d'en manipuler le contenu et les styles.

Voici le code

2 - Les objets node ou les noeuds du DOM

Le terme noeud est un terme générique qui sert à désigner tous les objets contenus dans le DOM.

A partir du noeud racine html, on voir que deux branches se forment.

3 - Alors que JavaScript est un langage de programmation qui permet d'opérer sur les objets DOM et de les manipuler, le DOM fournira les méthodes et les propriétés nécessaires pour lire, modifier mettre à jour et supprimer des éléments du document examiné. Par exemple, vous pouvez récupérer la valeur d'un champ texte HTML dans une chaîne à l'aide du DOM.

Exemples :

  • getElementbyId('coucou')
  • getElementsbyClassName('hello')
  • getElementsbyTagName('p')
  • querySelector('#diapo')
