Comme Twitter -

Un script qui contrôle que nous avons écrit 140 caractères et pas plus

Consignes

1

L'utilisateur peut saisir un texte de 140 caractères dans un textarea #tweetContenu.

Un paragraphe #compteur affiche les caractères restants.

2

  1. Détecter l'événement à chaque fois que l'utilisateur presse une touche
  2. Dans la fonction lancée lorsque l'"event" est détecté, calculer le nombre de caractères restants et l'afficher dans le compteur

Indications

  • Utiliser l'event keyup (cherchez à quoi il peut bien servir).
  • Utiliser la propriété elt.value (elt = élément) pour récupérer la valeur d'un input ou d'un textarea.
  • Utiliser .length pour compter le nombre de caractèresd'une chîne de caractères.
  • Utiliser elt.textContent=''; pour remplacer le contenu d'un élément HTML.
  • La sélection d'un élément avec querySelector - getElementbyId
  • Le gestionnaire d'événements avec addEventListener
  • La manipulation du DOM avec innerHTML - classList
