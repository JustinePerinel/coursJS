Rectangle

Construire un programme permettant d'afficher ou de cacher un rectangle à l'aide d'un bouton. Le rectangle doit changer d ecouleur lorsque l'on passe la souris dessus.

HTML

  • Dans l'en-tête de la page : ajoutez un titre au programme "Gestion de la souris
  • Créez un bouton (id : #toggle-rectangle) ayant pour intitulé "Cacher / Afficher le bouton"
  • Ajoutez des paragrapghes de texte expliquant ce que le programme peut faire
  • Créer une balise div vide (class : .rectangle) pour le rectangle

CSS

  • Le rectangle est bleu ('royalblue') de tailler 400x300px et possède des coins arrondis/
  • Lorsqu'on applique la classe "good" en plus au rectangle, il devient vert ('limegreen').
  • Lorsqu'on applique la classe "important" en plus au rectangle, il devient rouge ('firebrick').
  • Le changement de couleur de rectangle doit s'animer
  • Ajouter le CSS nécessaire restant pour que le prgramme fonctionne.

Intéractions

  • Quand on clique sur le bouton #toggle-rectangle, cela cache ou affiche le rectangle.
  • Quand la souris se déplace à l'intérieur du rectangle, il devient rouge. Il reprend sa couleur originale quand la souris n'est plus dedans.
  • Quand on doucle-clique sur le rectangle, il devient vert.

Remarques

  • Organiser le code en trois partie : les variables, les fonctions et enfin, le code principal.
  • Nommer correctement les gestionnaires d'événements pour qu'ils aient un sens.
