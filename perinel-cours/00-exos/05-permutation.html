Permutation de deux variables

Écrivez un programme '05-permutation.js' ayant initialement le contenu suivant
var nombre1 = 5;
var nombre2 = 3;
// tapez votre code ici sans rien modifier d'autre !
console.log(nombre1); // vous devez trouver 3
console.log(nombre2); // vous devez trouver 5
A l'endroit indiqué par un commentaire, ajouter le code nécessaire pour inverser les variables.

Il existe plusieurs solutions à cet exercice.
Vous n'êtes pas limité à l'utilisation de deux variables.

Résultat du programme


var temp;
temp = nombre1;
nombre1 = nombre2;
nombre2 = temp;
