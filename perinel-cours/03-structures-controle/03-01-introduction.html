III. 01 - Structures de contrôle

1 - Les condititons et les boucles

Il existe deux types de contrôle de structures de base, que l'on retrouve dans tous les langages de programmation et donc en JS. Ces deux types de structures de base sont :

  • Les structures de contrôle conditionnelles, aussi appellées les conditions.
  • Les structures de contrôle de boucles, aussi appelées les boucles.

Les conditions vont nous permettre d'exécuter un certain nombre d'instructions, si et seulement si, une certaine condition est vérifiée. Les boucles vont nous permettre d'exécuter un bloc de code en boucle tant qu'une condition donnée est vérifiée.

A - Les conditions

  • La condition simples :
    if... ;
    "SI x>5 affiche ça"
  • La condition un peu plus complexe :
    if... else... ;
    "SI x>5 affiche ça, SINON affiche ça"
  • La condition multiple :
    if... else if... else... ;
    "SI x>5 affiche ça SINON SI x<5 affiche ça SINON affiche ça".

Dans le cas où la coomparaison est vérifiée, le JavaScript renvoie le booléen true. Dans le cas contraire, le booléen false est renvoyé.

Le JavaScript va automatiquement comparé la valeur à gauche de l'opérateur à celle de droite selon l'opérateur de comparaison fourni et renvoyer le booléen true si la comparaison est validée ou false si la comparaison ne l'est pas. Exemples :

  • En sachant que let x = 4;, lorsque l'on inscrit le code suivant sur la page js liée :
    let testEgalValeur = x == 4;
    document.getElementById('p1').innerHTML = 'La comparaison selon laquelle x = 4 est ' + testEgalValeur + '.';
