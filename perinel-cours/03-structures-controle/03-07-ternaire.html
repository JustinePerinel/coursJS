III. 07 L'opérateur ternaire

Qu'est ce que l'opérateur ternaire ?

Les structures conditionnelles ternaires (aussi appellées ternaires) sont une autre façon d'écrire les conditions en utilisant une syntaxe basée sur l'opérateur ternaire : "question?soit ça : soit ça". Cet opérateur va nous permettre d'éécrire la condition sur une seule ligne. Les ternaires se présentent sous la forme suivante : test ? code à exécuter si true : code à exécuter si false.

1 - Exemple d'utilisation

  • Méthode if... else... pour let x = 15;
    if(x >= 10){
    document.getElementById('p2').innerHTML = 'x ('+x+') est supérieur à 10';
    }else{
    document.getElementById('p2').innerHTML = 'x ('+x+') est inférieur à 10';
    }

  • Méthode ternaire pour let x = 15;
    document.getElementById('p1').innerHTML =
    x >= 10 ? 'x ('+x+') est supérieur ou égal à 10' : 'x ('+x+') est strictement inférieur à 10';

  • Méthode if... else... pour let y = -20;
    if(y > 10){
    document.getElementById('p3').innerHTML = 'y ('+y+') est supérieur ou égal à 10';
    }else{
    document.getElementById('p3').innerHTML = 'y ('+y+') est inférieur à 10';
    }

  • Méthode ternaire pour let y = -20;
    document.getElementById('p4').innerHTML =
    y > 10 ? 'y ('+y+') est supérieur à 10' : 'y ('+y+') est inférieur à 10';
