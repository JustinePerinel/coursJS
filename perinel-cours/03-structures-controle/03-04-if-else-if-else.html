III. 04 - If... else if... else...

1 - If... else if... else... : explications

La condition if... else if... else ... est une structure conditionnelle encore plus complexe puisqu'elle permet de générer et de prendre en charge autant de cas que l'on le souhaite.

Nous allons pouvoir écrire autant de else if que l'on veut dans cette condition.

let x = 0.5;

if(x > 1){
alert('x contient une valeur strictement supérieure à 1');
}else if (x == 1){
alert('x contient la valeur 1');
}else{
alert('x contient une valeur strictement inférieure à 1');
}
Ici, le code va demander au navigateur d'effectuer les choses suivantes : "SI x qui contient la valeur 0.5 est supérieur à 1 lance cette première alerte SINON SI x est égal à 1, lis cette deuxième alerte, SINON lis la dernière alerte. C'est cette dernière qui va se décclancher.

Multiples conditions

Dès qu'un test va être validé, le JS va ignoré les tests suivants : il faut donc faire bien attention à l'odre des else if...lors de l'écriture d'une condition pour obtenir le résultat souhaité.

x = -10;

if(x > 1){
alert('x contient une valeur strictement supérieur à 1');
}else if (x == 1){
alert('x contient la valeur 1');
}else if (x < 1){ //C'est cette alerte qui sera lue car elle est vraie //
alert('x contient une valeur strictement inférieure à 1');
}else if (x < 0){
alert('x contient une valeur strictement inférieure à 0');
}else if (x < -2){
alert('x contient une valeur strictement inférieure à -2');
}else{
alert('la valeur de x n\'a pas pu être comparée à un nombre.')
}
Ici, le code l'alerte la plus proche et donc la plus logique aurait été l'avant-dernière et c'est pourtant au deuxième else if que le code va s'interrompre.
