III. 03 - If... else...

1 - If... else... : explication

La structure conditionnelle if... else... (en français : si... sinon...) sera plus complète car on va exécuter un bloc de code si un test renvoie true ou un autre dans le cas contraire.

Des exemples

  • Ici, on demande au navigateur : "SI la variable x qui contient la valeur 500 est plus grande que 1 affiche cette première string SINON affiche cette deuxième string."
    if(x > 1){
    alert('x contient une valeur supérieure à 1');
    }else{
    alert('x contient une valeur inférieure à 1');
    }
  • Ici, on demande au navigateur : "SI la variable y qui contient la valeur 0.5 est plus grande que 20 affiche cette première string SINON affiche cette deuxième string."
    if(y > 20){
    alert('y contient une valeur supérieure à 1');
    }else{
    alert('y contient une valeur inférieure à 1');
    }
