III. 05 - Les opérateurs logiques

Que sont-ils ?

Ce type d'opérateurs va nous permettre d'effectuer plusieurs comparaisons dans nos tests ou d'inverser la valeur logique du résultat d'un test.

1 - Présentation et utilisation des opérateurs logiques

Le JS supporte trois opérateurs logiques : l'opérateur logique "ET" ou &&, l'opérateur "OU" ou || et enfin l'opérateur "NON" ou !.

Opérateurs logiques
Nom des opérateurs Symbole des opérateurs Description
And (ET) && Renvoie true si toutes les comparaisons sont évaluées à true, ou false dans le cas inverse
Or (OU) || Renvoie true si au moins l'une des comparaisons est évaluée à true ; sinon c'est false.
No (NON) ! Renvoie false si une comparaison est évaluée à true et renvoie true dans le cas contraire


  • ET &&
    let x = 5;

    if(x >= 0 && x <=10){
    document.getElementById('p1').innerHTML = 'x qui vaut ' + x + ' est une valeur comprise entre 0 et 10';
    }else{
    document.getElementById('p1').innerHTML = 'x ne contient pas une valeur comprise entre 0 et 10';
    }

    Ici, on demande au navigateur :"SI x est supérieur ou égal à zéro ET SI x est inférieur ou égal à zéro, affiche cette première phrase, SINON, affiche la seconde.

  • OU ||
    if(y < -10 || y > 10){
    document.getElementById('p2').innerHTML = 'y qui vaut ' + y + ' est une valeur plus petite que -10 ou plus grande que 10.';
    }else{
    document.getElementById('p2').innerHTML = 'y qui vaut ' + y + ' n\'est pas une valeur plus petite que -10 ou plus grande que 10.';
    }

    Ici, on demande au navigateur :"SI y est inférieur à -10 OU SI y est supérieur à 10, affiche cette première phrase, SINON, affiche la seconde.

  • NON !
