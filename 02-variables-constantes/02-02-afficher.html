II. 02 - Des exemples de variables

1. Deux exemples de variables

Sur cette page, nous avons vu deux types de variables et la façon dont elles s'écrivent.

A - Let, sur une ligne ou deux

let permet de déclarer une variable sur le bloc auquel il appratient. On peut déclarer la variable de deux façons :

  • a. let variable
    variable = ''(chaîne de caractères ou nombre)
  • b. let variable = ''
  • Exemple
  • let ville = 'Suresnes'

B - Alert, faire apparaître une fenêtre d'alerte

Lorsque l'on a déclaré une variable ou que l'on veut faire apparaître une alerte sur la page web, il est possible de la déclarer en JavaScript. 2 solutions existent alors:

  • a. alert(prenom), lorsque l'on a déclaré auparavant une variable comme suit : let prenom = 'Justine', l'alerte affichera à ce moment là le contenu de la variable prenom, c'est-à-dire Justine.
  • b. alert('Bienvenue chez les Ch\'tis'), les apostrophes autour de la phrase permettent d'indiquer au navigateur que l'on veut qu'ill afficher ce qui se trouve dans les parebthèses.

2. Console log, afficher le contenu des variables dans l'inspecteur

Dans une page de JS, console.log(prenom) va permettre d'afficher le contenu de la variable indiquée entre parenthèse dans la partie console de l'inspecteur. On peut indiquer plusieurs variables dans le console.log en les séparant pas une virgule et un espace.
